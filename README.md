# 动物园小程序 动物园管理系统 （个人独立编写，不是网上资源）
## 一、介绍（获取源码，[查看文章末尾](#四-咨询我们)） 
计算机毕业设计 动物园小程序 动物园管理系统，前后端分离，基于node+vue项目，有登录、订票，天气查询，购买物品等功能
